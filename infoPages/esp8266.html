ESP8266 Wifi Tool

This tool can scan for WiFi devices to deauthenticate, block selected connections, and create dozens of spoof networks.
This project was a lot of fun to play with, I did not develop this from scratch, I used open source code as a starting point
I started with an ESP8266 which is a microporocessor that has a wifi card which allows it to send and receive packets on a network.
